Integrated C# Application with a Secure SQL Database System

Apr 2024 – Aug 2024

Designed and developed a comprehensive Relational Database System and integrated C# desktop application to manage the data for an E-Space solution, ensuring data integrity in the database.

Key Responsibilities & Deliverables:

  • • Database Design & Modeling: Created a robust data structure using an Entity-Relationship Diagram (ERD) and implemented Database Normalization (up to 3NF) to eliminate redundancy and ensure data integrity.
  • • Database Implementation (Back-end): Developed the database schema using SQL Data Definition Language (DDL) and populated it with realistic test data through SQL Data Manipulation Language (DML) queries (INSERT statements), with role-specific permissions to control data manipulation.
  • • Application Development (Front-end): Built a functional desktop application using C# and Microsoft Visual Studio 2022. The application served as the user interface, enabling secure, authenticated access to perform CRUD operations (Create, Read, Update, Delete) on the database.
  • • Querying & Reporting: Developed complex SQL Data Manipulation Language (DML) queries (e.g., joins, subqueries, aggregate functions) to support data retrieval and reporting functionalities within the application.
